## 🎯 Tema Escolhido
API - Database de Tênis (Quality Assurance)

## 📝 Descrição Inicial
Uma API RESTful desenvolvida em Python (Flask/FastAPI) focada no gerenciamento e consulta de um catálogo de tênis esportivos e casuais. O sistema foi projetado para facilitar a jornada de compra online, permitindo que consumidores e administradores busquem produtos com precisão. 

A API garante que o cliente só consiga visualizar e adquirir numerações que realmente estejam disponíveis no estoque. Entre as funcionalidades principais estão a busca por modelos, marcas e valores, além de filtros avançados para informações específicas do produto (cor, tamanho, lançamento e gênero). O projeto também inclui uma suíte de testes de Quality Assurance validando as regras de negócio através de requisições JSON utilizando Postman e Pytest.
