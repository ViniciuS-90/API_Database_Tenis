## 🎯 Tema Escolhido

## 📝 Descrição Inicial
Uma API RESTful desenvolvida em Python (Flask) focada no gerenciamento e consulta de um catálogo de tênis. A API garante que o usuario consiga visualizar modelos específicos de tênis retornando diversos valores como preço de modelos, lançamento, marca e imagens.
